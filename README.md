# Music Academy Management System (Demo)

System developed with Django for managing the operations of a music academy.

This repository contains a **sanitized demo version** of a real-world project,
with sensitive data, credentials, and company-specific information removed.

## Features
- Student management
- Class and instrument management
- Scheduling system
- Attendance tracking
- Invoicing module

## Technologies
- Python
- Django
- SQLite

## Notes
This demo version is intended for portfolio and educational purposes only.
